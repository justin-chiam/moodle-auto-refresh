# Moodle Auto Refresh
Browser extension that automatically refreshes Moodle tabs in the background every 10 minutes to prevent being signed out. Built for UNSW Moodle (moodle.telt.unsw.edu.au).

Works on both Firefox and Chrome.

## Features
- Automatically refreshes Moodle pages every 10 minutes
- Runs in the background using the alarms API
- Lightweight and simple, with no setup required

## Privacy
This extension does not collect, store, or transmit any user data.
